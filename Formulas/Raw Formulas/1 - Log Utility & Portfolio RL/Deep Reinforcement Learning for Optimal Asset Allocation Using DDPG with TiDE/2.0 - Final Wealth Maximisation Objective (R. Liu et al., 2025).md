The terminal objective ties all previous formulations together. It expresses the agent's goal in its most complete form: 
- Maximize the expected log of terminal wealth, which is the sum of all per-period reward signals added to the log of initial capital.
$$
\max\; \mathbb{E}\!\left[\ln W_0 + \sum_{t=1}^{T} \text{Reward}_t\right] = \max\; \mathbb{E}\!\left[\ln W_T\right]
$$

Where,

| $\max$                | Maximisation operator over the agent's policy.                                                                                       |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| $\mathbb{E}[\cdot]$   | Expectation over market outcomes.                                                                                                    |
| $\ln W_0$             | Log of initial wealth — a constant that does not affect the optimal policy but anchors the scale.                                    |
| $\sum_{t=1}^{T}$      | Summation over all trading periods.                                                                                                  |
| $\text{Reward}_t$     | Per-period reward signal, defined as $\gamma \ln(1 + r_t)$ (see [[1.3.1 - Logarithmic Reward (R. Liu et al., 2025)]]).               |
| $\ln W_T$             | Log of terminal wealth — the quantity whose expectation the agent maximises.                                                         |
| $T$                   | Total number of trading periods.                                                                                                     |

This equation ensures the agent prioritizes **sustainable capital growth**. 
- By optimizing for the log of terminal wealth, the agent naturally avoids catastrophic drawdowns, as the logarithmic function heavily penalizes returns that approach zero 
	- (i.e., $\ln(x) \to -\infty$ as $x \to 0$). 
- The equivalence between the left and right sides confirms that summing per-period log-rewards is a lossless decomposition of the terminal wealth objective
	- This makes the problem tractable for RL algorithms that operate on per-step reward signals. 
	- This convergence point unifies the 
		- **CRRA utility derivation** (see [[1.2 - CRRA Utility Function (R. Liu et al., 2025)]])
		- **Reward design** (see [[1.3.1 - Logarithmic Reward (R. Liu et al., 2025)]])
		- **Additive-multiplicative bridge** (see [[1.3.2 - Sum of Rewards to Wealth Identity (R. Liu et al., 2025)]]) 
	- All into a single optimization target.
