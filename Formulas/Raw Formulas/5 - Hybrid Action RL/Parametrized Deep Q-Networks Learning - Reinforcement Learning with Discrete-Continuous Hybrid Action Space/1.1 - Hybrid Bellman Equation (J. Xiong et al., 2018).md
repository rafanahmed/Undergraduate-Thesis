The classical Bellman optimality equation assumes actions are drawn from a single homogeneous space. The **Hybrid Bellman Equation** extends this to the hybrid action space (see [[1.0 - Hybrid Action Space (J. Xiong et al., 2018)]]) by introducing a two-level optimization: for each discrete action $k$, the continuous parameter $x_k$ is optimized via $\sup_{x_k \in X_k}$, and then the best discrete action is selected via $\max_{k \in [K]}$. This nested structure is the mathematical encoding of the hierarchical decision: *what* to do (discrete) and *how* to do it (continuous).

$$
Q(s_t, k_t, x_{k_t}) = \mathbb{E}_{r_t, s_{t+1}}\left[r_t + \gamma \max_{k \in [K]} \sup_{x_k \in X_k} Q(s_{t+1}, k, x_k) \;\middle|\; s_t = s, a_t = (k_t, x_{k_t})\right]
$$

Where,

| $Q(s_t, k_t, x_{k_t})$ | Action-value function for state $s_t$, discrete action $k_t$, and continuous parameter $x_{k_t}$ (see [[1.0 - Hybrid Action Space (J. Xiong et al., 2018)]]). |
| ------------------------ | -------------------------------------------------------------------------------------------------------------- |
| $s_t$                    | State at timestep $t$.                                                                                         |
| $k_t$                    | Discrete action selected at timestep $t$.                                                                      |
| $x_{k_t}$               | Continuous parameter associated with discrete action $k_t$ at timestep $t$.                                    |
| $r_t$                    | Immediate reward observed after taking action $(k_t, x_{k_t})$ in state $s_t$.                                 |
| $\gamma$                 | Discount factor, $\gamma \in [0, 1]$.                                                                          |
| $\max_{k \in [K]}$      | Outer maximization over the discrete action set — selects the best action type.                                 |
| $\sup_{x_k \in X_k}$    | Inner supremum over the continuous parameter space — finds the best parameterization for each $k$.              |
| $s_{t+1}$                | Next state, sampled from the transition distribution $p(\cdot | s_t, k_t, x_{k_t})$.                           |

The **Hybrid Bellman Equation** is the theoretical cornerstone of P-DQN. The inner $\sup$ is computationally intractable for a neural network Q-function (since $Q(s, k, x_k; \omega)$ is non-convex in $x_k$), but the right-hand side becomes tractable once $x_k^* = \arg\sup Q$ is provided. This observation motivates the introduction of the deterministic policy network $x_k(\cdot; \theta)$ (see [[1.1.1 - Deterministic Policy Approximation (J. Xiong et al., 2018)]]) to approximate $x_k^*$, reducing the problem to a standard discrete $\max$ over $K$ Q-values.
