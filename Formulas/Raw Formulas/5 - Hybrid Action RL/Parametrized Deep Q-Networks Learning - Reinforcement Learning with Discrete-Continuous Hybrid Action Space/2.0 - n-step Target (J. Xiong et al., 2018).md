The **n-step target** extends the standard one-step bootstrap target by accumulating $n$ observed rewards before bootstrapping from the Q-network. This reduces reliance on the Q-estimate (lowering bias from bootstrapping) at the cost of increased variance from the accumulated stochastic rewards. The n-step target uses the Deterministic Policy Approximation (see [[1.1.1 - Deterministic Policy Approximation (J. Xiong et al., 2018)]]) to evaluate the bootstrapped Q-value at the state reached after $n$ transitions.

$$
y_t = \sum_{i=0}^{n-1} \gamma^i r_{t+i} + \gamma^n \max_{k \in [K]} Q(s_{t+n}, k, x_k(s_{t+n}; \theta_t); \omega_t)
$$

Where,

| $y_t$                                               | The n-step target value at timestep $t$ — the regression target for the Q-network.                              |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| $\sum_{i=0}^{n-1} \gamma^i r_{t+i}$                | Discounted sum of the next $n$ observed rewards.                                                                 |
| $\gamma$                                             | Discount factor, $\gamma \in [0, 1]$.                                                                           |
| $n$                                                  | Number of lookahead steps before bootstrapping.                                                                  |
| $r_{t+i}$                                           | Reward observed at timestep $t + i$.                                                                             |
| $\max_{k \in [K]}$                                  | Maximization over discrete action types at the bootstrapped state (see [[1.0 - Hybrid Action Space (J. Xiong et al., 2018)]]). |
| $Q(s_{t+n}, k, x_k(s_{t+n}; \theta_t); \omega_t)$ | Q-value at state $s_{t+n}$ using the policy network's parameter output (see [[1.1.1 - Deterministic Policy Approximation (J. Xiong et al., 2018)]]). |
| $s_{t+n}$                                           | State reached after $n$ transitions from $s_t$. Set to terminal (Q-value = 0) if the episode ends within the window. |
| $\omega_t$, $\theta_t$                              | Current weights of the Q-network and policy network at step $t$.                                                 |

The **n-step target** introduces a fundamental trade-off between convergence speed and off-policy validity. When $n = 1$, the target reduces to the standard one-step bootstrap and the algorithm remains **off-policy**, enabling experience replay. When $n > 1$, the intermediate rewards $r_{t+1}, \ldots, r_{t+n-1}$ were collected under a potentially stale policy, making the algorithm **on-policy** in practice. The asynchronous P-DQN (Algorithm 2) uses $n$-step returns with $t_{\max}$ set between 2 and 20 depending on the domain, trading theoretical purity for faster convergence in delayed-reward settings.
