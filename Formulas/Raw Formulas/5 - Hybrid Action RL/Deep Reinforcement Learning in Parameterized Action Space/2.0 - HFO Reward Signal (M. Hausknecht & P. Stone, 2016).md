In the Half Field Offense domain, the true reward (winning a game) is far too sparse for an RL agent to gain learning traction. The paper introduces a *dense, hand-crafted* reward signal with four components that guide the agent through a behavioural curriculum: approaching the ball, kicking it toward the goal, and scoring. The weighted composition ensures that each phase of the task produces a positive gradient signal.

$$
r_t = d_{t-1}(a, b) - d_t(a, b) + I_t^{\text{kick}} + 3\left(d_{t-1}(b, g) - d_t(b, g)\right) + 5 \cdot I_t^{\text{goal}}
$$

Where,

| $r_t$              | Scalar reward at timestep $t$.                                                                                              |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| $d_t(a, b)$        | Euclidean distance between the agent $a$ and ball $b$ at timestep $t$.                                                      |
| $d_{t-1}(a, b)$    | Distance between agent and ball at the previous timestep.                                                                   |
| $I_t^{\text{kick}}$ | Indicator variable ($\in \{0, 1\}$) that fires once per episode when the agent first enters kicking range.                 |
| $d_t(b, g)$        | Euclidean distance between the ball $b$ and the center of the goal $g$ at timestep $t$.                                    |
| $d_{t-1}(b, g)$    | Distance between ball and goal center at the previous timestep.                                                             |
| $I_t^{\text{goal}}$ | Indicator variable ($\in \{0, 1\}$) for scoring a goal.                                                                    |
| $3$                | Weighting coefficient for the kick-to-goal component, compensating for the negative move-to-ball signal after each kick.    |
| $5$                | Terminal bonus magnitude for scoring.                                                                                       |

The **HFO Reward Signal** provides the dense training signal necessary for DDPG learning in the parameterized action space. Its multi-component structure creates an implicit behavioural curriculum: the move-to-ball component drives early exploration, the kick-to-goal component emerges once the agent can reliably approach, and the goal bonus reinforces successful scoring. The coefficient of 3 on the kick-to-goal component is a critical design choice — immediately after each kick, the move-to-ball component produces *negative* rewards as the ball moves away from the agent, and the kick-to-goal weighting must be large enough to overcome this conflicting signal.
