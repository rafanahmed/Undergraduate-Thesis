The target networks $Q'$ and $\mu'$ are updated to slowly track the online networks $Q$ and $\mu$ at each training step. This **Polyak averaging** approach replaces the periodic hard copies used in DQN, providing smoother target evolution.

$$
\theta^{Q'} = \tau\theta^Q + (1 - \tau)\theta^{Q'}, \quad \theta^{\mu'} = \tau\theta^\mu + (1 - \tau)\theta^{\mu'}
$$

Where,

| $\theta^{Q'}$  | Parameters of the target critic network.                                                |
| --------------- | --------------------------------------------------------------------------------------- |
| $\theta^Q$     | Parameters of the online critic network.                                                |
| $\theta^{\mu'}$ | Parameters of the target actor network.                                                |
| $\theta^\mu$   | Parameters of the online actor network.                                                 |
| $\tau$          | Soft update rate ($\tau \ll 1$; the paper uses $\tau = 10^{-4}$).                      |

The **Target Network Soft Updates** ensure that the regression targets used in the stabilized critic loss (see [[4.0 - Stabilized Critic Loss (M. Hausknecht & P. Stone, 2016)]]) change slowly and predictably. With $\tau = 10^{-4}$, the target network effectively averages over the last $\sim 10{,}000$ online network states, providing a stable reference point. This smooth tracking avoids the discontinuities inherent in periodic hard copies and is a standard component of the DDPG algorithm.
