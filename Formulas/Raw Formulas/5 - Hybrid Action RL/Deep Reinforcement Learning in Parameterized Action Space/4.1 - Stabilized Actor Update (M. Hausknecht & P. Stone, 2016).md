The stabilized version of the actor update takes expectations over states sampled from the replay memory $D$, averaging the critic's gradient signal across diverse experience rather than relying on the most recent transition.

$$
\nabla_{\theta^\mu} \mu = \mathbb{E}_{s_t \sim D}\left[\nabla_a Q(s_t, a|\theta^Q) \nabla_{\theta^\mu} \mu(s_t)\big|_{a = \mu(s_t)}\right]
$$

Where,

| $\nabla_{\theta^\mu} \mu$                | Expected policy gradient over the replay memory distribution.                                               |
| ---------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| $D$                                      | Replay memory (see [[4.0 - Stabilized Critic Loss (M. Hausknecht & P. Stone, 2016)]]).                     |
| $\nabla_a Q(s_t, a|\theta^Q)$           | Critic's gradient with respect to action, evaluated at the actor's current output (see [[3.1 - Actor Policy Gradient (M. Hausknecht & P. Stone, 2016)]]). |
| $\nabla_{\theta^\mu} \mu(s_t)$          | Jacobian of the actor network at state $s_t$.                                                              |
| $a = \mu(s_t)$                           | The action is evaluated at the actor's current deterministic output for state $s_t$.                        |

The **Stabilized Actor Update** averages gradient information across the replay buffer, providing a more representative gradient estimate than a single-sample update. Combined with the stabilized critic loss (see [[4.0 - Stabilized Critic Loss (M. Hausknecht & P. Stone, 2016)]]), this ensures that both the value function and the policy improve on a consistent, low-variance signal.
