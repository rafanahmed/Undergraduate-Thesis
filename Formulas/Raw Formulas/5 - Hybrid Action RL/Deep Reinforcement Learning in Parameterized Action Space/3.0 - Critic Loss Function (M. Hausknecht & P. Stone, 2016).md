The critic network $Q(s, a|\theta^Q)$ learns the action-value function by minimizing the temporal difference error. In continuous action spaces, the standard $\max_{a'} Q(s', a')$ operation is intractable, so the actor network provides the next-state action $a' = \mu(s'|\theta^\mu)$. This produces a regression target that the critic fits via gradient descent on the squared error.

$$
L_Q(s, a|\theta^Q) = \left(Q(s, a|\theta^Q) - \left(r + \gamma Q(s', \mu(s'|\theta^\mu)|\theta^Q)\right)\right)^2
$$

Where,

| $L_Q$                        | Critic loss function.                                                                                     |
| ---------------------------- | --------------------------------------------------------------------------------------------------------- |
| $Q(s, a|\theta^Q)$          | Critic network output: estimated Q-value for state-action pair $(s, a)$, parameterized by $\theta^Q$.     |
| $s$                          | Current state.                                                                                            |
| $a$                          | Current action (discrete action + continuous parameters from the actor).                                  |
| $r$                          | Observed reward (see [[2.0 - HFO Reward Signal (M. Hausknecht & P. Stone, 2016)]]).                      |
| $\gamma$                     | Discount factor.                                                                                          |
| $s'$                         | Next state observed after taking action $a$ in state $s$.                                                 |
| $\mu(s'|\theta^\mu)$        | Actor network output: the deterministic action proposed for next state $s'$, parameterized by $\theta^\mu$. |
| $\theta^Q$                   | Parameters (weights) of the critic network.                                                               |
| $\theta^\mu$                 | Parameters (weights) of the actor network.                                                                |

The **Critic Loss Function** is the standard DDPG loss adapted from the temporal difference framework. The accuracy of the critic is fundamental to the learning process because the actor's updates depend entirely on the quality of the critic's gradients. If the critic's Q-estimates are inaccurate, the gradients $\nabla_a Q$ point the actor in wrong directions, producing cascading errors. This interdependence motivates the stabilization techniques (see [[4.0 - Stabilized Critic Loss (M. Hausknecht & P. Stone, 2016)]]).
