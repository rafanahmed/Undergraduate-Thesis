The paper's core algorithmic contribution: a *continuous, non-saturating* gradient bounding technique that scales the critic's gradient by the parameter's relative distance from the relevant boundary. When the gradient suggests increasing $p$, the scaling factor measures how far $p$ is from $p_{\max}$; when the gradient suggests decreasing, it measures distance from $p_{\min}$. If the parameter exceeds bounds, the scaling factor becomes negative, *inverting* the gradient and actively pushing the parameter back.

$$
\nabla p = \nabla p \cdot \begin{cases} \dfrac{p_{\max} - p}{p_{\max} - p_{\min}} & \text{if } \nabla p \text{ suggests increasing } p \\[8pt] \dfrac{p - p_{\min}}{p_{\max} - p_{\min}} & \text{otherwise} \end{cases}
$$

Where,

| $\nabla p$                                      | The critic's gradient with respect to continuous parameter $p$ (see [[5.0 - Zeroing Gradients (M. Hausknecht & P. Stone, 2016)]]). |
| ------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| $p$                                              | Current activation (value) of the continuous parameter.                                                                             |
| $p_{\min}$                                       | Lower bound of the parameter's valid range.                                                                                         |
| $p_{\max}$                                       | Upper bound of the parameter's valid range.                                                                                         |
| $(p_{\max} - p) / (p_{\max} - p_{\min})$        | Scaling factor when increasing: approaches $0$ as $p \to p_{\max}$, becomes negative if $p > p_{\max}$.                            |
| $(p - p_{\min}) / (p_{\max} - p_{\min})$        | Scaling factor when decreasing: approaches $0$ as $p \to p_{\min}$, becomes negative if $p < p_{\min}$.                            |

**Inverting Gradients** is the central innovation that enables stable DDPG learning in bounded parameterized action spaces. Unlike squashing functions ($\tanh$), which saturate at boundaries and trap the parameter in a fixed extreme, the inverting approach maintains gradient *responsiveness*: when the critic reverses its recommendation, the opposite scaling factor provides full gradient magnitude for the return direction. Unlike zeroing (see [[5.0 - Zeroing Gradients (M. Hausknecht & P. Stone, 2016)]]), the gradient is smoothly attenuated rather than abruptly killed, avoiding discontinuities. The observed dash power reaches 98.8 out of 100 — respecting the boundary without saturation. This technique is **domain-agnostic** and applies to any bounded-continuous action space.
