The basic critic loss (see [[3.0 - Critic Loss Function (M. Hausknecht & P. Stone, 2016)]]) is prone to destructive feedback: errors in the Q-estimates propagate through the actor, which produces worse training data for the critic. The stabilized version introduces target networks $Q'$ and $\mu'$ (slow-moving replicas) and a replay memory $D$ (uniform sampling from a FIFO experience buffer). Together, these decouple the regression target from the rapidly changing online networks.

$$
L_Q(\theta^Q) = \mathbb{E}_{(s_t, a_t, r_t, s_{t+1}) \sim D}\left[\left(Q(s_t, a_t) - \left(r_t + \gamma Q'(s_{t+1}, \mu'(s_{t+1}))\right)\right)^2\right]
$$

Where,

| $L_Q(\theta^Q)$  | Stabilized critic loss, averaged over mini-batches from replay memory.                                    |
| ----------------- | --------------------------------------------------------------------------------------------------------- |
| $D$               | Replay memory: a FIFO queue of the agent's latest transitions $(s_t, a_t, r_t, s_{t+1})$.               |
| $Q(s_t, a_t)$    | Online critic's Q-value estimate for the sampled transition (see [[3.0 - Critic Loss Function (M. Hausknecht & P. Stone, 2016)]]). |
| $Q'$              | Target critic network: a slow-moving replica of $Q$ used to generate stable regression targets.          |
| $\mu'$            | Target actor network: a slow-moving replica of $\mu$ used to propose next-state actions.                 |
| $r_t$             | Observed reward at timestep $t$ (see [[2.0 - HFO Reward Signal (M. Hausknecht & P. Stone, 2016)]]).     |
| $\gamma$          | Discount factor.                                                                                          |
| $s_{t+1}$         | Next state.                                                                                               |

The **Stabilized Critic Loss** replaces the online networks in the TD target with their target counterparts, ensuring the regression target changes slowly. Mini-batch sampling from replay memory further reduces correlation between consecutive updates. This stabilized form is essential for reliable DDPG training — without it, the actor-critic feedback loop is prone to divergence. The target networks track the online networks via soft updates (see [[4.2 - Target Network Soft Updates (M. Hausknecht & P. Stone, 2016)]]).
