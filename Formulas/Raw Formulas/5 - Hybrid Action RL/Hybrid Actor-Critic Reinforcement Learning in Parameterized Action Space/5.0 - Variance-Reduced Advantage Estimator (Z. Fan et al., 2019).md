The variance-reduced advantage estimator computes an empirical estimate of the advantage function $A(s_t, a_t)$ from a trajectory segment of length $T$. It uses the state-value function $V(s)$ from the critic network as a baseline, combining observed rewards with a bootstrap estimate at the segment's end. This estimator is the bridge between the critic's value predictions and the sub-actors' policy updates — it is the shared signal that coordinates the discrete and continuous actor heads in the hybrid architecture.

$$
\hat{A}_t = -V(s_t) + r_t + \gamma r_{t+1} + \cdots + \gamma^{T-t-1} r_{T-1} + \gamma^{T-t} V(s_T)
$$

Where,

| $\hat{A}_t$                              | Estimated advantage at timestep $t$ within the trajectory segment.                              |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------- |
| $V(s_t)$                                  | State-value estimate from the critic network at the current timestep $t$.                       |
| $r_t, r_{t+1}, \ldots, r_{T-1}$          | Observed rewards along the trajectory segment from timestep $t$ to $T-1$.                       |
| $\gamma$                                  | Discount factor, $\gamma \in [0, 1)$.                                                          |
| $T$                                       | Length of the trajectory segment. Much less than the full episode length.                        |
| $V(s_T)$                                  | Bootstrap value estimate from the critic at the end of the trajectory segment.                  |

The advantage estimator can be written compactly as $\hat{A}_t = G_t^{(T)} - V(s_t)$, where $G_t^{(T)} = \sum_{i=0}^{T-t-1} \gamma^i r_{t+i} + \gamma^{T-t} V(s_T)$ is the $k$-step return. The parameter $T$ controls the **bias-variance tradeoff**: small $T$ gives low variance but high bias (relying heavily on $V(s_T)$'s accuracy); large $T$ gives lower bias but higher variance (accumulating stochastic reward signals). This estimator follows the implementation of Mnih et al. (2016) and is shared by both the discrete and continuous sub-actors — both actors receive the same advantage signal and are pushed toward actions that improve the same value criterion. This shared signal, along with the shared state encoding network, is the mechanism through which the independently-optimized sub-actors achieve coordinated behavior.
