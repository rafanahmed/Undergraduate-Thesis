The stochastic policy gradient theorem provides the gradient of the expected cumulative reward $J(\pi_\theta)$ with respect to the policy parameters $\theta$. Rather than computing values and deriving a policy from them (as in Q-learning), this approach *directly* optimizes the policy by computing how small changes in $\theta$ affect the expected return. The gradient is expressed as an expectation over states and actions, weighted by the action-value function.

$$
\nabla_\theta J(\pi_\theta) = \mathbb{E}_{s,a}\left[\nabla_\theta \log \pi_\theta(a \mid s)\, Q^{\pi_\theta}(s, a)\right]
$$

Where,

| $\nabla_\theta J(\pi_\theta)$              | Gradient of the expected policy value with respect to the policy parameters $\theta$.                                                          |
| ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| $\pi_\theta(a \mid s)$                      | Stochastic policy. Probability of taking action $a$ in state $s$ under parameterization $\theta$.                                              |
| $\nabla_\theta \log \pi_\theta(a \mid s)$   | Score function. The gradient of the log-probability of the selected action with respect to $\theta$.                                           |
| $Q^{\pi_\theta}(s, a)$                      | Action-value function under the current policy $\pi_\theta$ (see [[1.0 - Bellman Equation (Z. Fan et al., 2019)]]).                            |
| $\mathbb{E}_{s,a}$                          | Expectation over the state distribution induced by $\pi_\theta$ and the action distribution $\pi_\theta(\cdot \mid s)$.                        |

The stochastic policy gradient is the foundation of **REINFORCE** and modern actor-critic methods. The score function $\nabla_\theta \log \pi_\theta(a \mid s)$ acts as a direction in parameter space that increases the probability of action $a$, weighted by $Q^{\pi_\theta}(s, a)$ which measures how good that action is. Actions with high Q-values are reinforced; actions with low Q-values are suppressed. However, using the raw Q-value as the weighting introduces high variance into the gradient estimate because $Q$ includes the baseline value of being in state $s$ regardless of action. The Advantage-Based Policy Gradient (see [[2.1 - Advantage-Based Policy Gradient (Z. Fan et al., 2019)]]) addresses this by subtracting the state-value baseline.
