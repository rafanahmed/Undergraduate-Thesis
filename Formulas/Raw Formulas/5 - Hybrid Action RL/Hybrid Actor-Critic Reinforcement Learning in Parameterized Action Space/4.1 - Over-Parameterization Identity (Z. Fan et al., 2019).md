The over-parameterization identity formalizes a structural flaw in action-value critics applied to parameterized action spaces. When the Q-function takes the state, the selected discrete action, and the continuous parameters for *all* discrete actions as input, the resulting function must learn to ignore all parameters except those corresponding to the selected action. The identity establishes that the true Q-value depends only on the parameter of the selected action $a$, rendering all other parameters $x_{a'}$ for $a' \neq a$ as irrelevant noise in the critic's input.

$$
Q(s, a, x_{a_1}, x_{a_2}, \ldots, x_{a_k}) = Q(s, a, x_a)
$$

Where,

| $Q$                                    | Action-value function in the parameterized action space (see [[1.0 - Bellman Equation (Z. Fan et al., 2019)]]).               |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| $s$                                     | Current state.                                                                                                                |
| $a$                                     | Selected discrete action from $A_d$ (see [[4.0 - Parameterized Action Space (Z. Fan et al., 2019)]]).                        |
| $x_{a_1}, x_{a_2}, \ldots, x_{a_k}$    | Continuous parameters for all $k$ discrete actions, output simultaneously by the continuous actor network.                    |
| $x_a$                                   | Continuous parameter for the *selected* action $a$ only.                                                                     |

This identity is the **core motivation** for the paper's architectural choice of a state-value critic $V(s)$ over an action-value critic $Q(s, a, x)$. When a Q-function critic is used, the network receives $\sum_i m_{a_i}$ extra input dimensions that are pure noise for any given action selection. The network must learn an internal *masking* operation conditioned on $a$ — an additional representational burden that degrades value estimation accuracy. The state-value function $V(s)$ sidesteps this problem entirely by conditioning only on the state, requiring no action or parameter information. The empirical impact of over-parameterization is most visible in the Moving environment, where P-DQN (which uses a Q-value critic) achieves only 1.56% success rate while H-PPO (state-value critic) achieves 90.45%.
