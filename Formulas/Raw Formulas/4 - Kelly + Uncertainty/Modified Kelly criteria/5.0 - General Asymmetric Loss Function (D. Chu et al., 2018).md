The **general asymmetric loss function** $l_3$ generalizes the absolute and squared error loss functions while introducing *asymmetric penalties* for overestimation versus underestimation of the Kelly fraction. It is motivated by the common practical complaint that the Kelly fraction is too large: by penalizing overestimation more heavily than underestimation, the resulting Bayes estimator is biased toward smaller, safer betting fractions.

$$
l_3(f, p) = (c_1 \mathbb{I}_{f > k(p)} + c_2) \, |f - k(p)|^k
$$

Where,

| $l_3(f, p)$             | Loss from using fraction $f$ when the true optimal is $k(p)$, under asymmetric penalty.                     |
| ------------------------ | ------------------------------------------------------------------------------------------------------------ |
| $f$                      | Estimated betting fraction, $f \in [0, 1]$.                                                                 |
| $k(p)$                   | [[1.1.1 - Kelly Criterion (D. Chu et al., 2018)]]. True optimal Kelly fraction.                             |
| $c_1$                    | Additional penalty coefficient for overestimation ($f > k(p)$), $c_1 > 0$.                                  |
| $c_2$                    | Base penalty coefficient, $c_2 > 0$.                                                                        |
| $k$                      | Exponent on the absolute deviation, $1 < k < 2$. Interpolates between absolute error and squared error.     |
| $\mathbb{I}_{f > k(p)}$ | Indicator function: equals $1$ when $f > k(p)$, $0$ otherwise.                                              |
| $\lvert \cdot \rvert$    | Absolute value.                                                                                              |

The loss function $l_3$ nests standard loss functions as special cases: $c_1 = 0, c_2 = 1, k = 1$ gives absolute error loss $l_1$; $c_1 = 0, c_2 = 1, k = 2$ gives squared error loss $l_2$. The paper considers two parameterizations: $(c_1, c_2, k) = (1, 1, 1.5)$ where overestimation is penalized double, and $(1, 2, 1.5)$ where overestimation is penalized 1.5 times. The Bayes estimator $f_3$ minimizes $G(f) = \int_0^1 l_3(f, p)\,\pi(p \mid x)\,dp$ (extending the framework of [[4.0 - Expected Posterior Loss (D. Chu et al., 2018)]]) and requires **numerical optimization** — the integral is computed by adaptive quadrature (splitting at the discontinuity in the indicator) and the minimum is found by brute-force search over $f$ in steps of $0.001$. In all examples, $f_{3a} < f_{3b} < f_0$, confirming that stronger overestimation penalties produce smaller fractions.
