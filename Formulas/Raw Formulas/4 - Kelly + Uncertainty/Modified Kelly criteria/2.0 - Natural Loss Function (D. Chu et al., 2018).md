The **natural loss function** $l_0$ measures the cost of using an estimated betting fraction $f$ instead of the true optimal Kelly fraction $k(p)$, quantified as the *difference in expected log growth*. Unlike generic loss functions (absolute error, squared error), this loss function is specifically calibrated to the Kelly context: it measures forgone growth in the units that matter — logarithmic wealth.

$$
l_0(f, p) = p \log\!\left(\frac{1 - k(p) + \theta k(p)}{1 - f + \theta f}\right) + (1 - p) \log\!\left(\frac{1 - k(p)}{1 - f}\right)
$$

Where,

| $l_0(f, p)$   | Loss incurred by using fraction $f$ instead of the optimal $k(p)$. Non-negative; zero iff $f = k(p)$.          |
| -------------- | --------------------------------------------------------------------------------------------------------------- |
| $f$            | Estimated betting fraction, $f \in [0, 1]$.                                                                    |
| $p$            | True (unknown) probability of winning (see [[1.0 - Profitability Condition (D. Chu et al., 2018)]]).            |
| $k(p)$         | [[1.1.1 - Kelly Criterion (D. Chu et al., 2018)]]. True optimal Kelly fraction.                                |
| $\theta$       | European (decimal) odds (see [[1.0 - Profitability Condition (D. Chu et al., 2018)]]).                          |
| $\log(\cdot)$  | Natural logarithm.                                                                                              |

The loss function $l_0$ is the **pivot** of the paper's Bayesian framework. It allows the authors to treat the betting fraction as a statistical estimator whose quality is evaluated against the Kelly desiderata, rather than using a generic statistical distance. Because the expected log growth (see [[1.1 - Expected Log Growth (D. Chu et al., 2018)]]) is concave and $k(p)$ is its unique maximizer, $l_0(f, p) \geq 0$ always, with equality iff $f = k(p)$. Critically, $l_0$ is **asymmetric around** $k(p)$: overestimation ($f > k(p)$) incurs greater loss than underestimation ($f < k(p)$) because the expected posterior loss $G(f)$ increases more steeply for $f > f_0$. This asymmetry is the mathematical mechanism that drives all modified Kelly fractions to be *conservative*. The loss function feeds into the expected posterior loss $G(f)$ (see [[4.0 - Expected Posterior Loss (D. Chu et al., 2018)]]).
