The **expected posterior loss** $G(f)$ is the Bayesian criterion function that the modified Kelly fractions minimize. It integrates the natural loss function $l_0(f, p)$ over the posterior distribution of $p$, producing a scalar measure of the expected cost of using fraction $f$ given the observed data and prior beliefs. The Bayes estimator is the value of $f$ that minimizes $G(f)$.

$$
G(f) = \int_0^1 l_0(f, p) \, \pi(p \mid x) \, dp
$$

Where,

| $G(f)$            | Expected posterior loss: the Bayesian cost of using fraction $f$ given data $x$ and prior $(a, b)$.         |
| ------------------ | ----------------------------------------------------------------------------------------------------------- |
| $l_0(f, p)$       | [[2.0 - Natural Loss Function (D. Chu et al., 2018)]]. Measures forgone expected log growth.                |
| $\pi(p \mid x)$   | Posterior density of $p$ given data $x$ (see [[3.0 - Beta Posterior Distribution (D. Chu et al., 2018)]]). |
| $f$                | Estimated betting fraction, $f \in [0, 1]$.                                                                |
| $p$                | Integration variable: the true (unknown) winning probability.                                               |

$G(f)$ is **convex** in $f$, with its minimum at $f_0$ (see [[4.1 - Modified Kelly Criterion (D. Chu et al., 2018)]]). The key simplification: because $f$ appears only in the *denominators* of the logarithms in $l_0$, minimizing $G(f)$ is equivalent to maximizing $Q(f) = \hat{p}\log(1 - f + \theta f) + (1 - \hat{p})\log(1 - f)$, which has the same form as the expected log growth (see [[1.1 - Expected Log Growth (D. Chu et al., 2018)]]) with the posterior mean $\hat{p}$ replacing $p$. This collapse from a complex integral to a function of a single sufficient statistic $\hat{p}$ is the paper's most elegant analytical result. The expected posterior loss increases more rapidly for $f > f_0$ than for $f < f_0$, reflecting the **asymmetric cost** of overbetting versus underbetting.
