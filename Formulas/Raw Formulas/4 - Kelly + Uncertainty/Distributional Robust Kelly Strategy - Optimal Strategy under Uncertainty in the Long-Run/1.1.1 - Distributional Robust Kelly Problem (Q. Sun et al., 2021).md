The **Distributional Robust Kelly Problem (DRKP)** is the max-min optimization that defines the robust Kelly strategy. The gambler chooses the bet allocation $\mathbf{b}$ to maximize the worst-case log growth rate (see [[1.1 - Worst-Case Log Growth Rate (Q. Sun et al., 2021)]]) over the uncertainty set $\Xi$. This formulation hedges against the **optimizer's curse** — the phenomenon where optimal decisions under estimated distributions disappoint out-of-sample.

$$
\max_{\mathbf{b} \in \mathcal{B}} \quad \inf_{\boldsymbol{\pi} \in \Xi} \mathbb{E}_\pi[\log(\mathbf{r}^\top \mathbf{b})]
$$

Where,

| $\max_{\mathbf{b} \in \mathcal{B}}$                | Maximize over all feasible bet allocations in the convex constraint set $\mathcal{B} \subseteq \mathcal{S}_n$.             |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| $\inf_{\boldsymbol{\pi} \in \Xi}$                  | Infimum over all distributions in the uncertainty set (see [[1.1 - Worst-Case Log Growth Rate (Q. Sun et al., 2021)]]).    |
| $\mathbb{E}_\pi[\log(\mathbf{r}^\top \mathbf{b})]$ | Expected log growth rate under distribution $\boldsymbol{\pi}$ (see [[1.0 - Mean Log Growth Rate (Q. Sun et al., 2021)]]). |
| $\mathcal{B}$                                      | Convex constraint set on the bet allocation; must satisfy $\mathcal{B} \subseteq \mathcal{S}_n$.                           |
| $\Xi$                                              | Uncertainty set of plausible distributions (see [[1.1 - Worst-Case Log Growth Rate (Q. Sun et al., 2021)]]).               |

The DRKP is convex (maximizing a concave function over a convex set) but **not automatically tractable**. The paper's central computational contribution is showing that for specific forms of $\Xi$ — polyhedral, box, ellipsoidal, $f$-divergence balls, Wasserstein balls, and moment-based sets — the DRKP can be reformulated as a **Disciplined Convex Programming (DCP)** problem solvable via CVXPY.
- The key technique is **Lagrangian duality**: express the inner $\inf$ as a minimization, form the dual, verify strong duality via Slater's condition, and show the resulting dual problem follows DCP rules.
- Concrete DCP forms include the box set (see [[2.0 - Box DRKP (Q. Sun et al., 2021)]]), the Wasserstein ball (see [[3.0 - Wasserstein Ball DRKP (Q. Sun et al., 2021)]]), and the $f$-divergence ball (see [[4.0 - f-Divergence Ball DRKP (Q. Sun et al., 2021)]]).
