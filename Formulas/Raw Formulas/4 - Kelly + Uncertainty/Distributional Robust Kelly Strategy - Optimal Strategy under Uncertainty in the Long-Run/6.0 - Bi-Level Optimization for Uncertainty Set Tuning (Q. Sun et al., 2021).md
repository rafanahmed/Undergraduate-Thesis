Constructing a high-quality uncertainty set $\Xi$ is the fundamental practical challenge for the DRKP (see [[1.1.1 - Distributional Robust Kelly Problem (Q. Sun et al., 2021)]]). This bi-level optimization automates the tuning of the uncertainty set parameters $\theta$ by maximizing out-of-sample log-growth performance. The inner problem solves the DRKP for a given $\theta$; the outer problem evaluates the resulting bet on held-out data. Because the inner DRKP is a DCP problem, the solution map $\mathbf{b}^*(\theta)$ is differentiable with respect to $\theta$ via CVXPYLayers (Agrawal et al., 2019), enabling gradient-based tuning through backpropagation.

$$
\begin{aligned}
\mathbf{b}^*(\theta) &= \arg\max_{\mathbf{b} \in \mathcal{B}} \min_{\boldsymbol{\pi} \in \Xi(\theta)} \mathbb{E}_\pi[\log(\mathbf{r}^\top \mathbf{b})] \\
\max_\theta \; L(\theta) &= \frac{1}{N} \sum_{i=1}^N \log(\mathbf{r}_i^\top \mathbf{b}^*(\theta))
\end{aligned}
$$

Where,

| $\mathbf{b}^*(\theta)$                | Optimal robust bet for uncertainty set parametrized by $\theta$ — the solution to the inner DRKP.                |
| --------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| $\theta$                                | Parameters of the uncertainty set $\Xi(\theta)$ (e.g., nominal distribution, radii, shape matrices).             |
| $\Xi(\theta)$                           | Parametrized uncertainty set — its shape and size depend on $\theta$.                                            |
| $L(\theta)$                             | Out-of-sample Kelly loss — the average log-growth of the robust bet evaluated on held-out observations.          |
| $N$                                     | Number of out-of-sample observations.                                                                            |
| $\mathbf{r}_i$                          | The $i$-th out-of-sample return observation.                                                                     |
| $\mathcal{B}$                           | Convex constraint set on bet allocation (see [[1.1.1 - Distributional Robust Kelly Problem (Q. Sun et al., 2021)]]). |

The bi-level structure resolves the **coverage-tightness tension** *data-adaptively*: the gradient $\nabla_\theta L(\theta)$ steers $\theta$ toward an uncertainty set that is neither too conservative (overly hedged) nor too narrow (failing to capture the true distribution).
- The differentiability of the solution map $\mathbf{b}^*(\theta)$ relies on the DCP tractability of the inner problem — the same property established in the paper's computational contributions.
- Implementation uses **CVXPYLayers**, which wraps CVXPY problems as differentiable PyTorch layers, enabling the entire pipeline to be trained end-to-end via backpropagation with optimizers like Adam.
