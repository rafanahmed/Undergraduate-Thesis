The Quadratic Risk-Constrained Kelly (QRCK) problem is a quadratic approximation of the full RCK problem (see [[3.0 - Risk-Constrained Kelly Gambling Problem (E. Busseti et al., 2016)]]), obtained by Taylor-expanding the objective and constraint about $\mathbf{r}^\top \mathbf{b} \approx 1$. Using the excess return $\boldsymbol{\rho} = \mathbf{r} - \mathbf{1}$, the approximations $\log(\mathbf{r}^\top \mathbf{b}) \approx \boldsymbol{\rho}^\top \mathbf{b} - \tfrac{1}{2}(\boldsymbol{\rho}^\top \mathbf{b})^2$ and $(\mathbf{r}^\top \mathbf{b})^{-\lambda} \approx 1 - \lambda \boldsymbol{\rho}^\top \mathbf{b} + \tfrac{\lambda(\lambda+1)}{2}(\boldsymbol{\rho}^\top \mathbf{b})^2$ produce a **convex quadratic program**. The QRCK solution is provably a **Markowitz portfolio** under the no-arbitrage assumption, establishing a formal link between the RCK framework and classical mean-variance optimization.

$$
\begin{aligned}
&\text{maximize} \quad E[\boldsymbol{\rho}^\top \mathbf{b}] - \frac{1}{2} E[(\boldsymbol{\rho}^\top \mathbf{b})^2] \\
&\text{subject to} \quad \mathbf{1}^\top \mathbf{b} = 1, \quad \mathbf{b} \geq 0, \\
&\phantom{\text{subject to} \quad} -\lambda E[\boldsymbol{\rho}^\top \mathbf{b}] + \frac{\lambda(\lambda+1)}{2} E[(\boldsymbol{\rho}^\top \mathbf{b})^2] \leq 0
\end{aligned}
$$

Where,

| $\boldsymbol{\rho}$                                                                | Excess return vector: $\boldsymbol{\rho} = \mathbf{r} - \mathbf{1}$, so $\mathbf{r}^\top \mathbf{b} = 1 + \boldsymbol{\rho}^\top \mathbf{b}$.                   |
| ----------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $E[\boldsymbol{\rho}^\top \mathbf{b}]$                                             | Expected excess return: $\boldsymbol{\mu}^\top \mathbf{b}$ where $\boldsymbol{\mu} = E[\boldsymbol{\rho}]$.                                                     |
| $E[(\boldsymbol{\rho}^\top \mathbf{b})^2]$                                         | Raw second moment of excess return: $\mathbf{b}^\top \mathbf{S} \mathbf{b}$ where $\mathbf{S} = E[\boldsymbol{\rho}\boldsymbol{\rho}^\top]$.                    |
| $\lambda$                                                                           | Risk-aversion parameter (see [[2.0 - Drawdown Risk Bound (E. Busseti et al., 2016)]]).                                                                          |
| $\mathbf{b}$                                                                        | Decision variable: the bet allocation vector.                                                                                                                    |

The QRCK problem is a **convex QP** that approximates the full RCK problem when returns are close to 1 (the financial portfolio regime). Its solution provides a useful warm-start for the primal-dual stochastic gradient method applied to the full RCK problem. Under the **no-arbitrage** assumption (any long-only portfolio with positive expected excess return has positive return variance), the QRCK solution is a **Markowitz portfolio** — it solves the classical mean-variance problem $\text{maximize } \boldsymbol{\mu}^\top \mathbf{b} - (\gamma/2) \mathbf{b}^\top \boldsymbol{\Sigma} \mathbf{b}$ for some $\gamma > 0$, where $\boldsymbol{\Sigma} = \mathbf{S} - \boldsymbol{\mu}\boldsymbol{\mu}^\top$ is the return covariance. This connection establishes that Markowitz optimization is a *special case* of the RCK framework — specifically, its second-order approximation.
