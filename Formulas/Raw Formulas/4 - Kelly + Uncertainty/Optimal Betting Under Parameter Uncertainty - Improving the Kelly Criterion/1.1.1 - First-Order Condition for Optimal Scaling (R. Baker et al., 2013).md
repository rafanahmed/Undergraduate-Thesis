The optimal shrinkage factor $k^*$ is the unique solution to the first-order condition obtained by differentiating the [[1.1 - Scaled Expected Utility (R. Baker et al., 2013)]] with respect to $k$ and setting equal to zero. This integral equation cannot be solved in closed form for general $f(q)$, but can be evaluated numerically using the **Beta Method** (assuming $f$ is a Beta distribution) or approximated analytically via Taylor expansion (see [[2.0 - Shrinkage Factor Approximation (R. Baker et al., 2013)]]).

$$
\int_0^1 f(q) \, s^*(q) \left\{ \frac{pb}{1 + bk^*s^*(q)} - \frac{1 - p}{1 - k^*s^*(q)} \right\} dq = 0
$$

Where,

| $f(q)$       | Sampling distribution pdf (see [[1.0 - Expected Maximized Utility Under Parameter Uncertainty (R. Baker et al., 2013)]]).                                                   |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $s^*(q)$     | Kelly-optimal bet at estimated probability $q$ (see [[1.0 - Expected Maximized Utility Under Parameter Uncertainty (R. Baker et al., 2013)]]).                              |
| $p$          | True winning probability.                                                                                                                                                   |
| $b$          | Fractional odds.                                                                                                                                                            |
| $k^*$        | **Optimal shrinkage factor**: the value of $k$ that maximizes expected utility in the [[1.1 - Scaled Expected Utility (R. Baker et al., 2013)]] integral.                   |

The equation is solved via **Newton-Raphson iteration**: $k_{n+1} = k_n - (dE(u^*)/dk) / (d^2E(u^*)/dk^2)$. For the **Beta Method**, $f(q)$ is taken as a Beta distribution with parameters $\alpha = p\{p(1-p)/\sigma^2 - 1\}$ and $\beta = (1-p)\{p(1-p)/\sigma^2 - 1\}$, where $\sigma^2$ is the variance of the probability estimate. This provides the most accurate numerical solution but requires specifying the functional form of the sampling distribution. The first-order approximation (see [[2.0 - Shrinkage Factor Approximation (R. Baker et al., 2013)]]) avoids this requirement by providing a closed-form expression valid for small $\sigma^2$.
