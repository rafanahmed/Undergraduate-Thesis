The expected log terminal wealth achieved by an investor who uses the **optimal Bayesian gambling control** $f_t^* = \hat{Z}_t / \sigma^2$ (see [[5.0 - Optimal Bayesian Gambling Control (Browne & Whitt, 1996)]]) over the horizon $[0, T]$. This value function decomposes into three economically interpretable components: the contribution of initial wealth, the growth from the known-mean component of the drift, and the net gain from the random drift component after accounting for the cost of learning.

$$
F^B(x) = \ln x + \frac{\mu^2}{2\sigma^2}T + \frac{c}{2\sigma^2}T - \frac{1}{2}\ln\!\left(1 + \frac{c}{\sigma^2}T\right)
$$

Where,

| $F^B(x)$                                              | Bayesian value function — the maximum expected log terminal wealth under the optimal Bayesian policy.            |
| ------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------- |
| $x$                                                    | Initial wealth at time zero.                                                                                     |
| $\ln x$                                                | Log contribution of initial wealth.                                                                              |
| $\frac{\mu^2}{2\sigma^2}T$                            | Growth from the known-mean component — identical to the non-Bayesian value (see [[2.2 - Continuous-Time Value Function (Browne & Whitt, 1996)]]). |
| $\frac{c}{2\sigma^2}T$                                | Potential growth from the random drift component — the additional value available if $Z$ were known (linear in $T$). |
| $\frac{1}{2}\ln\!\left(1 + \frac{c}{\sigma^2}T\right)$ | **Learning penalty** — the cost of having to estimate $Z$ from noisy observations (logarithmic in $T$).       |
| $c$                                                    | Prior variance of the unobserved drift $Z$.                                                                      |
| $\sigma^2$                                             | Known variance rate.                                                                                             |
| $T$                                                    | Investment horizon.                                                                                              |

The value function reveals the paper's central economic insight: the potential gain from a random drift is $O(T)$ (linear), while the learning penalty is $O(\ln T)$ (logarithmic). As $T \to \infty$, the learning penalty becomes negligible relative to the gain, meaning the Bayesian investor **asymptotically captures the full value** of having a random drift. The ratio of captured value to potential value is $1 - \frac{\ln(1 + cT/\sigma^2)}{cT/\sigma^2}$, which approaches $1$. The difference between this value function and the non-Bayesian baseline and the perfect-information benchmark yields the **Financial Value of Learning** (see [[5.1.1 - Financial Value of Learning (Browne & Whitt, 1996)]]).
