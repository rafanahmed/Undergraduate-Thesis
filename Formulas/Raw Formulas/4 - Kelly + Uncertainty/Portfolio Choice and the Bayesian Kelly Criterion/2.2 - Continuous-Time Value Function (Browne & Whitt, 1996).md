The expected log terminal wealth at horizon $T$ when the investor uses the optimal **continuous-time Kelly fraction** $f^* = \mu/\sigma^2$ (see [[2.1 - Continuous-Time Kelly Fraction (Browne & Whitt, 1996)]]) and the drift $\mu$ is a known constant. This is the non-Bayesian benchmark — the best an investor can achieve when the drift is known with certainty. It serves as the baseline for measuring both the **financial value of randomization** and the **cost of learning** in the Bayesian framework.

$$
F(x) = \ln x + \frac{T}{2}\frac{\mu^2}{\sigma^2}
$$

Where,

| $F(x)$    | Value function — the maximum expected log terminal wealth starting from initial wealth $x$.                     |
| ---------- | ---------------------------------------------------------------------------------------------------------------- |
| $x$        | Initial wealth at time zero.                                                                                     |
| $\ln x$    | Contribution of initial wealth to log terminal wealth.                                                           |
| $T$        | Investment horizon (terminal time).                                                                              |
| $\mu$      | Known drift parameter of the risky opportunity.                                                                  |
| $\sigma^2$ | Known variance rate.                                                                                             |
| $\mu^2 / \sigma^2$ | Squared Sharpe ratio — the fundamental measure of risk-adjusted opportunity quality.                    |

The growth rate $\mu^2/(2\sigma^2)$ is half the squared Sharpe ratio and represents the *rate* at which log-wealth accumulates under optimal Kelly investing. The value function is **linear in $T$** — there is no diminishing return to a longer horizon when parameters are known. This linearity contrasts with the Bayesian case (see [[5.1 - Bayesian Value Function (Browne & Whitt, 1996)]]), where the learning penalty introduces a logarithmic correction. The difference $F^B(x) - F(x)$ quantifies the **gain from Bayesian learning** over a naive investor who ignores the random drift component (see [[5.1.1 - Financial Value of Learning (Browne & Whitt, 1996)]]).
