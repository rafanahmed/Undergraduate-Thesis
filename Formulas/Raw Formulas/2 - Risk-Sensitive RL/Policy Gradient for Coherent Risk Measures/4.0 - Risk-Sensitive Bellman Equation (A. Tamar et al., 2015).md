The risk-sensitive Bellman equation defines the value function $V_\theta(x)$ for the dynamic Markov coherent risk problem. It is the dynamic programming equation that decomposes the infinite-horizon risk $\rho_\infty(M)$ into a recursive relationship: the value at state $x$ equals the immediate cost plus a discounted *worst-case expectation* of the value at the next state, where the worst case is taken over the risk envelope $\mathcal{U}$ (see [[1.0 - Representation Theorem (A. Tamar et al., 2015)]]).

This equation is the convergence point of the Representation Theorem's risk-as-optimization view and the Markov coherent risk's recursive structure (see [[2.0 - Markov Coherent Risk (A. Tamar et al., 2015)]]). By Ruszczyński (2010, Theorem 1), the operator defined by this equation has a unique fixed point $V_\theta$, which equals the risk objective: $V_\theta(x_0) = \rho_\infty(M)$.

$$
V_\theta(x) = C(x) + \gamma \max_{\xi P_\theta(\cdot|x) \,\in\, \mathcal{U}(x,\, P_\theta(\cdot|x))} \mathbb{E}_\xi\!\left[V_\theta(x')\right]
$$

Where,

| $V_\theta(x)$                          | Risk-sensitive value function at state $x$ under policy $\theta$. Equals $\rho_\infty(M \mid x_0 = x)$.                                               |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| $C(x)$                                  | Bounded deterministic cost at state $x$, with $C(x) \in [-C_{\max}, C_{\max}]$.                                                                       |
| $\gamma$                                | Discount factor, $\gamma \in (0, 1)$.                                                                                                                  |
| $\xi$                                   | Test density function over next states (see [[1.0 - Representation Theorem (A. Tamar et al., 2015)]]).                                                 |
| $P_\theta(\cdot\|x)$                   | Transition probability from state $x$ under policy $\theta$: $P_\theta(x'\|x) = \sum_a P(x'\|x, a)\mu_\theta(a\|x)$.                                 |
| $\mathcal{U}(x, P_\theta(\cdot\|x))$   | State-dependent risk envelope induced by the transition probability at $x$ (see [[1.1 - Risk Envelope (A. Tamar et al., 2015)]]).                      |
| $\mathbb{E}_\xi[V_\theta(x')]$         | $\xi$-weighted expectation of the value function at the next state $x'$.                                                                                |
| $x'$                                    | Next state, drawn from $P_\theta(\cdot\|x)$.                                                                                                           |
| $\max$                                  | Adversarial maximization: the "environment" chooses the density $\xi$ from the envelope that maximizes the expected future value (i.e., the worst case for costs). |

The key difference from the risk-neutral Bellman equation is the **inner maximization** over the risk envelope. Instead of a simple expectation $\mathbb{E}[V_\theta(x')]$, the equation uses $\max_\xi \mathbb{E}_\xi[V_\theta(x')]$ — the agent plans as if the environment adversarially re-weights transition probabilities within the envelope. This establishes the formal equivalence with **robust MDPs** (Osogami, 2012): the risk-sensitive value function is identical to the value function of a robust MDP with uncertainty set $\mathcal{U}(x, P_\theta(\cdot|x))$.

Under the contraction condition (Assumption E.2: $\xi(x') \leq \kappa/\gamma$ for all feasible $\xi$), the risk-sensitive Bellman operator $T_\theta[V](x) := C_\theta(x) + \gamma \max_\xi \mathbb{E}_\xi[V]$ is a contraction, guaranteeing that the projected value iteration (PRSVI) algorithm converges to the unique fixed point $V_\theta$.

The value function $V_\theta$ is the object approximated by the **critic** in the actor-critic algorithm, and its gradient is the target of the **actor**. The stage-wise cost function $h_\theta$ (see [[4.1 - Stage-wise Cost Function (A. Tamar et al., 2015)]]) and the Dynamic Policy Gradient Theorem (see [[4.1.1 - Dynamic Policy Gradient Theorem (A. Tamar et al., 2015)]]) are both derived from this equation.
