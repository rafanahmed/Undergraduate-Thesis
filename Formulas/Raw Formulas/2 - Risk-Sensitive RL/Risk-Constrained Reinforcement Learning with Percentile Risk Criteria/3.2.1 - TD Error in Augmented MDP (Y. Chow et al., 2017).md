The **temporal-difference (TD) error** in the augmented MDP $\bar{M}$ is the one-step prediction error of the critic's linear value function approximation. It measures the discrepancy between the observed augmented cost plus the discounted estimated future value and the current value estimate. This scalar signal drives both the critic update (to improve the value function approximation) and the actor update (as a proxy for the advantage function). Combined with the score function $\nabla_\theta \log \mu(a_k \mid x_k, s_k; \theta)$, it provides an unbiased estimate of the Lagrangian's policy gradient (see [[3.1 - Augmented Policy Gradient Theorem (Y. Chow et al., 2017)]]).

$$
\delta_k(v_k) = \bar{C}_{\lambda_k}(x_k, s_k, a_k) + \gamma\,v_k^\top \phi(x_{k+1}, s_{k+1}) - v_k^\top \phi(x_k, s_k)
$$

Where,

| $\delta_k(v_k)$                       | TD error at step $k$, computed using critic weight vector $v_k$.                                                                                         |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $\bar{C}_{\lambda_k}(x_k, s_k, a_k)$ | Augmented cost at step $k$ (see [[3.0 - Augmented MDP Cost Identity (Y. Chow et al., 2017)]]). Depends on the current Lagrange multiplier $\lambda_k$.  |
| $\gamma$                                | Discount factor.                                                                                                                                          |
| $v_k$                                   | Critic weight vector at step $k$.                                                                                                                         |
| $\phi(x_k, s_k)$                       | Feature vector for augmented state $(x_k, s_k)$, $\phi \in \mathbb{R}^{\kappa_1}$.                                                                       |
| $\phi(x_{k+1}, s_{k+1})$              | Feature vector at the next augmented state.                                                                                                               |
| $v_k^\top \phi(x, s)$                  | Linear value function approximation: $\tilde{V}^{\theta, v}(x, s) \approx V^\theta(x, s)$.                                                               |

The TD error plays three roles in the algorithm. First, it drives the **critic update**: $v_{k+1} = v_k + \zeta_4(k)\,\delta_k(v_k)\,\phi(x_k, s_k)$, which is the standard linear TD(0) update on the augmented state space, converging to the projected Bellman fixed point (see [[3.2 - Bellman Operator on Augmented MDP (Y. Chow et al., 2017)]]). Second, it drives the **actor update**: $\theta_{k+1} = \Gamma_\Theta(\theta_k - \frac{\zeta_2(k)}{1-\gamma}\nabla_\theta \log \mu(a_k \mid x_k, s_k; \theta)\cdot\delta_k(v_k))$, providing a low-variance incremental gradient estimate. Third, the TD error implicitly encodes the CVaR constraint through the augmented cost $\bar{C}_\lambda$, which charges the CVaR penalty at the target state. The critic operates on the **fastest timescale** ($\zeta_4$) in the four-timescale stochastic approximation scheme, so from the perspective of all other updates, the value function approximation is quasi-stationary.
