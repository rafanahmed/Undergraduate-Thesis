The sub-gradient of the Lagrangian $L(\nu, \theta, \lambda)$ with respect to the VaR parameter $\nu$ provides the descent direction for the $\nu$-update. It is a *sub-gradient* rather than a gradient because the positive part function $(Z - \nu)^+$ is non-differentiable at $Z = \nu$. The sub-gradient involves the probability that the constraint cost exceeds the current VaR estimate, which connects directly to the definition of VaR as the $\alpha$-quantile of $J^\theta(x_0)$.

$$
\partial_\nu L(\nu, \theta, \lambda) \ni \lambda\!\left(1 - \frac{1}{1 - \alpha}\,P(J^\theta(x_0) \geq \nu)\right)
$$

Where,

| $\partial_\nu L$          | Sub-differential of the Lagrangian with respect to $\nu$ (see [[2.0 - CVaR Lagrangian Function (Y. Chow et al., 2017)]]). A set-valued mapping because $(Z - \nu)^+$ has a kink at $Z = \nu$. |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $\ni$                      | Set membership: the right-hand side is one element of the sub-gradient set $\partial_\nu L$.                                                                                                     |
| $\lambda$                  | Lagrange multiplier (see [[2.0 - CVaR Lagrangian Function (Y. Chow et al., 2017)]]).                                                                                                            |
| $\alpha$                   | CVaR confidence level.                                                                                                                                                                           |
| $P(J^\theta(x_0) \geq \nu)$ | Tail probability: the probability that the cumulative constraint cost exceeds $\nu$ under policy $\theta$.                                                                                     |
| $J^\theta(x_0)$           | Cumulative discounted constraint cost (see [[1.1 - CVaR-Constrained MDP (Y. Chow et al., 2017)]]).                                                                                              |
| $\nu$                      | VaR parameter (see [[1.0 - CVaR Minimization Formula (Y. Chow et al., 2017)]]).                                                                                                                 |

At the optimal $\nu^* = \text{VaR}_\alpha(J^\theta(x_0))$, the sub-gradient contains zero: $P(J^\theta(x_0) \geq \nu^*) = 1 - \alpha$ (for continuous distributions), so $\lambda(1 - (1-\alpha)^{-1}(1-\alpha)) = 0$. The sub-gradient is estimated in the trajectory-based PG algorithm using the indicator $\mathbf{1}\{J(\xi) \geq \nu\}$ as a sample estimate of $P(J^\theta(x_0) \geq \nu)$. In the actor-critic setting, this estimator can only be evaluated at the end of a trajectory (when $s_{\text{Tar}}$ is observed), which limits incrementality. The **SPSA-based** alternative approximates the sub-gradient using finite-difference perturbations of the critic's value function, enabling fully incremental updates at the cost of introducing bias that vanishes as the perturbation $\Delta_k \to 0$. The convergence analysis handles the non-differentiability via **differential inclusions** (Borkar, 2008, Chapter 5).
