The **Bellman operator** $B^\theta$ on the augmented MDP $\bar{M}$ defines the fixed-point equation for the value function of the CVaR Lagrangian problem. Its fixed point $V^* = B^\theta[V^*]$ satisfies $V^*(x_0, \nu) = L(\nu, \theta, \lambda) - \lambda\nu + \lambda\beta$ (up to terms independent of $\theta$). The operator inherits the **contraction property** from the discount factor $\gamma < 1$, which guarantees a unique fixed point and enables TD-learning convergence for the critic. The projected fixed point $\tilde{V}^* = \Pi B^\theta[\tilde{V}^*]$ is the critic's target when using linear function approximation.

$$
B^\theta[V](x, s) = \sum_{a \in \mathcal{A}} \mu(a \mid x, s;\, \theta)\!\left[\bar{C}_\lambda(x, s, a) + \gamma\!\sum_{x', s'} \bar{P}(x', s' \mid x, s, a)\,V(x', s')\right]
$$

Where,

| $B^\theta[V](x, s)$                    | Bellman operator applied to value function $V$ at augmented state $(x, s)$.                                                                                                                             |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $\mu(a \mid x, s;\, \theta)$           | Policy on the augmented MDP, conditioned on both original state $x$ and budget state $s$.                                                                                                                |
| $\bar{C}_\lambda(x, s, a)$             | Augmented cost (see [[3.0 - Augmented MDP Cost Identity (Y. Chow et al., 2017)]]): $C(x, a)$ for $x \in \mathcal{X}'$, and $\lambda(-s)^+/(1-\alpha)$ for $x = x_{\text{Tar}}$.                       |
| $\gamma$                                 | Discount factor.                                                                                                                                                                                         |
| $\bar{P}(x', s' \mid x, s, a)$        | Augmented transition: $P(x' \mid x, a)\,\mathbf{1}\{s' = (s - D(x, a))/\gamma\}$ for $x \in \mathcal{X}'$, and $\mathbf{1}\{x' = x_{\text{Tar}}, s' = 0\}$ for $x = x_{\text{Tar}}$.                |
| $V(x', s')$                             | Value function evaluated at the next augmented state.                                                                                                                                                    |
| $\mathcal{A}$                            | Action space.                                                                                                                                                                                            |

The Bellman operator has the standard form of a policy-evaluation operator, but on the *augmented* state space. **Theorem 10** establishes that the critic update $v_{k+1} = v_k + \zeta_4(k)\phi(x_k, s_k)\delta_k(v_k)$ converges to $v^*$, the minimizer of the projected Bellman residual $\|B^\theta[\Phi v] - \Phi v\|_{d_\gamma^\theta}^2$, under the assumption that basis functions are linearly independent (Assumption 9) and samples are drawn from $\pi_\gamma^\theta$. The contraction property ensures that all eigenvalues of the matrix $A$ in the projected TD fixed-point equation have positive real part (Lemma 20), guaranteeing a unique stable equilibrium for the critic ODE $\dot{v} = b - Av$. The TD error from this operator provides the unbiased gradient estimate used in the actor update (see [[3.2.1 - TD Error in Augmented MDP (Y. Chow et al., 2017)]]).
