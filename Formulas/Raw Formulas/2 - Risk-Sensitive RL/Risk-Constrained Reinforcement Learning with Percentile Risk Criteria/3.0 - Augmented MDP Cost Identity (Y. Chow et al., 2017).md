The **augmented MDP** $\bar{M}$ is the central construction that transforms the CVaR Lagrangian problem into a standard (unconstrained) MDP. The augmented state space $\bar{\mathcal{X}} = \mathcal{X} \times \mathcal{S}$ appends an auxiliary state $s$ that tracks the remaining "constraint budget" — starting at $s_0 = \nu$ and decreasing by $D(x_k, a_k)/\gamma^k$ at each step. The cost identity below establishes that the total discounted cost in $\bar{M}$ equals the Lagrangian's $\theta$-dependent terms: the expected primary cost plus the CVaR penalty. This convergence point merges the Lagrangian relaxation (see [[2.0 - CVaR Lagrangian Function (Y. Chow et al., 2017)]]) with the state augmentation motivated by Bäuerle and Ott (2011), who showed that the optimal CVaR-constrained policy depends on the current state, time step, and accumulated constraint cost.

$$
\sum_{k=0}^{T} \gamma^k \bar{C}_\lambda(x_k, s_k, a_k) = G^\theta(x) + \frac{\lambda}{1 - \alpha}\,(J^\theta(x) - s)^+
$$

Where,

| $\bar{C}_\lambda(x_k, s_k, a_k)$ | Augmented cost function: equals $C(x_k, a_k)$ for transient states $x_k \in \mathcal{X}'$, and $\lambda(-s_k)^+/(1-\alpha)$ at the target state $x_{\text{Tar}}$. |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $\gamma$                            | Discount factor, $\gamma \in (0, 1)$.                                                                                                                               |
| $x_k$                              | State of the original MDP at step $k$.                                                                                                                               |
| $s_k$                              | Auxiliary "budget" state: $s_0 = \nu$, with transition $s_{k+1} = (s_k - D(x_k, a_k))/\gamma$.                                                                     |
| $a_k$                              | Action at step $k$, drawn from $\mu(\cdot \mid x_k, s_k; \theta)$.                                                                                                  |
| $G^\theta(x)$                      | Cumulative discounted primary cost: $\sum_{k=0}^{T-1} \gamma^k C(x_k, a_k)$.                                                                                        |
| $\lambda$                           | Lagrange multiplier (see [[2.0 - CVaR Lagrangian Function (Y. Chow et al., 2017)]]).                                                                                |
| $\alpha$                            | CVaR confidence level.                                                                                                                                               |
| $J^\theta(x)$                      | Cumulative discounted constraint cost: $\sum_{k=0}^{T-1} \gamma^k D(x_k, a_k)$.                                                                                     |
| $s$                                 | Initial budget state, set to $\nu$ at the start of each trajectory.                                                                                                  |
| $(\cdot)^+$                        | Positive part operator.                                                                                                                                              |

This identity is the **key enabling result** for the actor-critic algorithms. It means the value function of $\bar{M}$ at initial state $(x_0, \nu)$ equals the Lagrangian's $\theta$-dependent component: $V^\theta(x_0, \nu) = \mathbb{E}[G^\theta(x_0)] + \frac{\lambda}{1-\alpha}\mathbb{E}[(J^\theta(x_0) - \nu)^+]$. Consequently, $\nabla_\theta L(\nu, \theta, \lambda) = \nabla_\theta V^\theta(x_0, \nu)$, which means the standard **policy gradient theorem** (Sutton et al., 2000) can be applied to $\bar{M}$ to obtain the Lagrangian's gradient (see [[3.1 - Augmented Policy Gradient Theorem (Y. Chow et al., 2017)]]). The augmented cost $\bar{C}_\lambda$ collects the CVaR penalty entirely at the terminal state $x_{\text{Tar}}$, where $(-s_{\text{Tar}})^+$ measures the total constraint budget overrun. Standard TD-learning and value function approximation can then be deployed on $\bar{M}$ via the Bellman operator (see [[3.2 - Bellman Operator on Augmented MDP (Y. Chow et al., 2017)]]).
