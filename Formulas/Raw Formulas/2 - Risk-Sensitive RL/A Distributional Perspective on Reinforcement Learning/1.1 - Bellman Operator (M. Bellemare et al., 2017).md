The Bellman operator $\mathcal{T}^\pi$ provides a recursive characterization of the value function $Q^\pi$ for a fixed policy $\pi$. Rather than computing the infinite sum of discounted rewards directly, it decomposes the value at any state-action pair into the *immediate expected reward* plus the *discounted expected future value* under the transition dynamics and policy. This operator is the backbone of policy evaluation algorithms like SARSA.

The Bellman operator is a **contraction mapping** in the $L_\infty$ norm with contraction factor $\gamma$, meaning repeated application converges exponentially to the unique fixed point $Q^\pi$.

$$
\mathcal{T}^\pi Q(x, a) := \mathbb{E}\left[R(x, a)\right] + \gamma\,\mathbb{E}_{P, \pi}\left[Q(x', a')\right]
$$

Where,

| $\mathcal{T}^\pi$              | Bellman operator for policy $\pi$. Maps a value function $Q$ to a new value function $\mathcal{T}^\pi Q$.                          |
| ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| $Q(x, a)$                      | Value function being evaluated (see [[1.0 - Value Function (M. Bellemare et al., 2017)]]).                                         |
| $\mathbb{E}\left[R(x,a)\right]$| Expected immediate reward from state-action pair $(x, a)$, where $R$ is a random variable.                                        |
| $\gamma$                        | Discount factor, $\gamma \in [0, 1]$.                                                                                              |
| $\mathbb{E}_{P, \pi}$          | Expectation over both the transition kernel $P(\cdot \mid x, a)$ and the policy $\pi(\cdot \mid x')$.                              |
| $x'$                            | Next state, drawn from $P(\cdot \mid x, a)$.                                                                                       |
| $a'$                            | Next action, drawn from $\pi(\cdot \mid x')$.                                                                                      |

The Bellman operator $\mathcal{T}^\pi$ is a $\gamma$-**contraction** in $\|\cdot\|_\infty$: for any two value functions $Q_1, Q_2$, $\|\mathcal{T}^\pi Q_1 - \mathcal{T}^\pi Q_2\|_\infty \le \gamma\|Q_1 - Q_2\|_\infty$. This guarantees that the iterates $Q_{k+1} := \mathcal{T}^\pi Q_k$ converge exponentially to $Q^\pi$ from any initialization.

The paper introduces a *distributional* analogue of this operator (see [[3.2 - Distributional Bellman Operator (M. Bellemare et al., 2017)]]) that applies the same recursive decomposition to the full return distribution rather than its expectation. The contraction property carries over to the distributional setting — but in the Wasserstein metric rather than $L_\infty$.
