The transition operator $P^\pi$ maps a value distribution $Z$ forward by one step under the dynamics of the MDP and a fixed policy $\pi$. Given a state-action pair $(x, a)$, it produces the distribution of the return at the *next* state-action pair $(X', A')$ — which is itself random because both the next state and the next action are stochastic.

This operator isolates the transition-mixing component of the distributional Bellman equation (see [[3.0 - Distributional Bellman Equation (M. Bellemare et al., 2017)]]). Combined with the reward shift and discount scaling, it forms the full distributional Bellman operator (see [[3.2 - Distributional Bellman Operator (M. Bellemare et al., 2017)]]).

$$
P^\pi Z(x, a) \stackrel{D}{:=} Z(X', A')
$$

Where,

| $P^\pi$                | Transition operator for policy $\pi$. Maps a value distribution $Z \in \mathcal{Z}$ to a new value distribution $P^\pi Z \in \mathcal{Z}$.    |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| $Z(X', A')$            | Return distribution at the random next state-action pair. A random distribution because $(X', A')$ is random.                                  |
| $X'$                    | Next state, drawn from the transition kernel: $X' \sim P(\cdot \mid x, a)$.                                                                   |
| $A'$                    | Next action, drawn from the policy: $A' \sim \pi(\cdot \mid X')$.                                                                             |
| $\stackrel{D}{:=}$     | Definitional equality in distribution.                                                                                                         |
| $\mathcal{Z}$          | Space of value distributions with bounded moments.                                                                                             |

The transition operator $P^\pi$ performs a **mixture** over the next-state return distributions, weighted by the transition probabilities and the policy. For a finite state space, $P^\pi Z(x, a)$ is a mixture distribution: $P^\pi Z(x, a) = \sum_{x', a'} P(x' \mid x, a)\,\pi(a' \mid x')\,Z(x', a')$.

In the contraction proof for $\mathcal{T}^\pi$ (Lemma 3), the key step is bounding $d_p(P^\pi Z_1(x, a), P^\pi Z_2(x, a)) \le \sup_{x', a'} d_p(Z_1(x', a'), Z_2(x', a'))$. This follows because mixing can only reduce Wasserstein distance — the mixture's quantile function is a weighted average of the components' quantile functions.
