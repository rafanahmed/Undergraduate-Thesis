To make distributional learning practical, the paper parametrizes the value distribution as a **discrete categorical distribution** over a fixed set of $N$ equally-spaced atoms. Each atom represents a possible return value, and the neural network outputs the probability assigned to each atom via a softmax layer. This parametrization replaces the single scalar Q-value output of DQN with an $N$-dimensional probability vector for each action.

The discrete representation has two key advantages: it is highly *expressive* (with enough atoms, it can approximate any distribution to arbitrary precision) and *computationally friendly* (the softmax output head integrates naturally into deep network architectures). The support bounds $V_{\text{MIN}}$ and $V_{\text{MAX}}$ are hyperparameters that impose an inductive bias by treating all returns beyond the range as equivalent.

$$
\begin{aligned}
z_i &= V_{\text{MIN}} + i\,\Delta z, \quad 0 \le i < N, \quad \Delta z = \frac{V_{\text{MAX}} - V_{\text{MIN}}}{N - 1} \\[6pt]
Z_\theta(x, a) &= z_i \;\;\text{w.p.}\;\; p_i(x, a) := \frac{e^{\theta_i(x,a)}}{\sum_{j=0}^{N-1} e^{\theta_j(x,a)}}
\end{aligned}
$$

Where,

| $z_i$                   | The $i$-th atom location in the discrete support. A fixed, predetermined return value.                                                  |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| $V_{\text{MIN}}$        | Lower bound of the return support. Set to $-10$ in experiments.                                                                         |
| $V_{\text{MAX}}$        | Upper bound of the return support. Set to $10$ in experiments.                                                                          |
| $\Delta z$              | Spacing between consecutive atoms: $(V_{\text{MAX}} - V_{\text{MIN}}) / (N - 1)$.                                                      |
| $N$                      | Number of atoms. The paper uses $N = 51$ for the final agent (C51).                                                                    |
| $i$                      | Atom index, $0 \le i < N$.                                                                                                              |
| $Z_\theta(x, a)$        | The parametric value distribution at state-action pair $(x, a)$, defined over the discrete support $\{z_i\}$.                           |
| $p_i(x, a)$             | Probability assigned to atom $z_i$ at $(x, a)$, produced by a softmax over the network's raw outputs.                                  |
| $\theta_i(x, a)$        | Raw (logit) output of the neural network for atom $i$ at state-action pair $(x, a)$.                                                   |
| $\theta$                 | The full set of neural network parameters: $\theta : \mathcal{X} \times \mathcal{A} \to \mathbb{R}^N$.                                 |

The parametrization introduces a fundamental algorithmic challenge: the Bellman update $\hat{\mathcal{T}}z_j = r + \gamma z_j$ shifts and scales the atoms, placing them at locations that generally do *not* coincide with the original support $\{z_i\}$. This necessitates the **projected Bellman update** (see [[4.1 - Projected Bellman Update (M. Bellemare et al., 2017)]]) which maps the updated distribution back onto the fixed support.

The choice of $N = 51$ atoms is not arbitrary. The paper demonstrates that performance monotonically increases with $N$ across the training games, with no evidence of network capacity saturation. Even the minimal $N = 2$ case (the **Bernoulli algorithm**) outperforms DQN in 3 of 5 training games, suggesting that any amount of distributional information is beneficial. The expected value used for greedy action selection is recovered as $Q(x, a) = \sum_{i=0}^{N-1} z_i\,p_i(x, a)$.
