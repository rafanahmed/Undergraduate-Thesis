The value function is the foundational quantity in reinforcement learning. It describes the *expected* cumulative discounted reward an agent receives from a given state-action pair $(x, a)$ when following a fixed policy $\pi$ thereafter. The value function $Q^\pi$ compresses the entire distribution of future returns into a single scalar — the mean — which traditional RL algorithms use as their sole learning target.

In the distributional framework, $Q^\pi(x, a)$ is reinterpreted as the expectation of a random variable $Z^\pi(x, a)$, the **random return**. This reframing is the starting point for the paper's argument that the full distribution of $Z^\pi$, not merely its mean, should be the central object of study.

$$
Q^\pi(x, a) := \mathbb{E}\left[Z^\pi(x, a)\right] = \mathbb{E}\left[\sum_{t=0}^{\infty} \gamma^t R(x_t, a_t)\right]
$$

Where,

| $Q^\pi(x, a)$ | Value function for policy $\pi$. The expected return from taking action $a$ in state $x$ and following $\pi$ thereafter. |
| -------------- | ------------------------------------------------------------------------------------------------------------------------ |
| $Z^\pi(x, a)$ | Random return under policy $\pi$. The random variable whose realization is the discounted sum of future rewards.         |
| $\mathbb{E}$  | Expectation operator.                                                                                                    |
| $\gamma$       | Discount factor, $\gamma \in [0, 1]$. Controls the trade-off between immediate and future rewards.                      |
| $R(x_t, a_t)$ | Reward received at time $t$, treated as a random variable.                                                               |
| $x_t$          | State at time $t$, with $x_t \sim P(\cdot \mid x_{t-1}, a_{t-1})$ and $x_0 = x$.                                       |
| $a_t$          | Action at time $t$, with $a_t \sim \pi(\cdot \mid x_t)$ and $a_0 = a$.                                                  |
| $\pi$          | A stationary policy mapping each state $x \in \mathcal{X}$ to a probability distribution over actions $\mathcal{A}$.     |

The value function $Q^\pi$ is the object that standard algorithms like SARSA and Q-Learning seek to learn. It satisfies a recursive relationship — the **Bellman equation** (see [[1.1 - Bellman Operator (M. Bellemare et al., 2017)]]) — which decomposes the expected return into the immediate expected reward plus the discounted expected future value.

The paper's central thesis is that collapsing the random return $Z^\pi$ to its mean $Q^\pi$ discards valuable distributional structure — multimodality, variance, tail behaviour — that can improve learning stability and representation quality. The value function remains important as the *expectation* of the value distribution, and greedy action selection in the proposed algorithm still operates over $Q^\pi(x, a) = \sum_i z_i\,p_i(x, a)$, but the learning target is the full distribution rather than the scalar mean.
