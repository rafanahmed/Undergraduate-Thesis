The categorical cross-entropy loss is the training objective for the C51 algorithm. It measures the KL divergence between the projected Bellman target distribution $\Phi\,\hat{\mathcal{T}}\,Z_{\tilde{\theta}}$ (see [[4.1 - Projected Bellman Update (M. Bellemare et al., 2017)]]) and the model's current predicted distribution $Z_\theta$. Since the projected target is treated as fixed (via the target network parameter $\tilde{\theta}$), minimizing the KL divergence reduces to minimizing the cross-entropy term.

This loss replaces the squared Bellman error $(r + \gamma Q(x', \pi(x')) - Q(x, a))^2$ used in DQN. The shift from regression on a scalar value to classification over a categorical distribution is the key algorithmic change that enables distributional learning within the DQN architecture.

$$
L_{x,a}(\theta) = D_{\text{KL}}\!\left(\Phi\,\hat{\mathcal{T}}\,Z_{\tilde{\theta}}(x, a) \;\big\|\; Z_\theta(x, a)\right) = -\sum_{i=0}^{N-1} m_i \log p_i(x, a) + \text{const.}
$$

Where,

| $L_{x,a}(\theta)$                                             | Sample loss for the transition at state-action pair $(x, a)$, as a function of the network parameters $\theta$.                                                                  |
| -------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| $D_{\text{KL}}(\cdot \| \cdot)$                               | Kullback-Leibler divergence. Measures the information-theoretic distance from $Z_\theta$ to the target $\Phi\,\hat{\mathcal{T}}\,Z_{\tilde{\theta}}$.                            |
| $\Phi\,\hat{\mathcal{T}}\,Z_{\tilde{\theta}}(x, a)$          | [[4.1 - Projected Bellman Update (M. Bellemare et al., 2017)]]. The projected Bellman target distribution, computed from the target network $\tilde{\theta}$.                    |
| $Z_\theta(x, a)$                                              | Current predicted distribution at $(x, a)$ (see [[4.0 - Categorical Distribution Parametrization (M. Bellemare et al., 2017)]]).                                                 |
| $m_i$                                                          | The $i$-th component of the projected target: $m_i = (\Phi\,\hat{\mathcal{T}}\,Z_{\tilde{\theta}}(x, a))_i$. Treated as a fixed target (no gradient flows through $m_i$).       |
| $p_i(x, a)$                                                   | Predicted probability for atom $z_i$ under the current network (see [[4.0 - Categorical Distribution Parametrization (M. Bellemare et al., 2017)]]).                             |
| $\tilde{\theta}$                                               | Target network parameters, held fixed and periodically copied from $\theta$ (following the DQN training regime).                                                                 |
| $N$                                                            | Number of atoms.                                                                                                                                                                  |

The cross-entropy loss between categorical distributions is one of the best-understood and most well-behaved losses in deep learning. It produces smooth, informative gradients even when the target and prediction differ substantially. The paper argues this contributes to the empirical success of C51 — the optimization landscape is significantly friendlier than the squared Bellman error used in standard DQN.

The KL divergence is *not* sensitive to the values of the atoms — it only depends on probability masses, not on where those masses sit on the number line. This stands in contrast to the Wasserstein metric (see [[2.0 - Wasserstein Metric (M. Bellemare et al., 2017)]]), which does respect the geometry of the return space. Despite this apparent mismatch with the theoretical analysis (which uses the Wasserstein metric), experiments on CliffWalk (Appendix D) demonstrate that the categorical projection with cross-entropy loss empirically minimizes the Wasserstein distance more effectively than direct Wasserstein loss minimization via stochastic gradient descent.
