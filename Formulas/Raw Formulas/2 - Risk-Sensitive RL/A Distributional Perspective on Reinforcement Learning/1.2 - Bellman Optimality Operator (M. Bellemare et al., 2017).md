The Bellman optimality operator $\mathcal{T}$ characterizes the *optimal* value function $Q^*$ — the maximum expected return achievable from any state-action pair by any policy. Unlike the policy evaluation operator (see [[1.1 - Bellman Operator (M. Bellemare et al., 2017)]]), which evaluates a fixed policy $\pi$, the optimality operator selects the *best* action at each successor state via the $\max$ operation. This operator underlies Q-Learning and its deep variants.

Like $\mathcal{T}^\pi$, the optimality operator $\mathcal{T}$ is a $\gamma$-contraction in $\|\cdot\|_\infty$ with unique fixed point $Q^*$. However, its distributional analogue is fundamentally different: the paper proves that the distributional optimality operator is **not** a contraction in any distributional metric.

$$
\mathcal{T}\,Q(x, a) := \mathbb{E}\left[R(x, a)\right] + \gamma\,\mathbb{E}_P\left[\max_{a' \in \mathcal{A}} Q(x', a')\right]
$$

Where,

| $\mathcal{T}$                                         | Bellman optimality operator. Maps a value function $Q$ to $\mathcal{T}\,Q$ by incorporating greedy action selection.                |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------- |
| $Q(x, a)$                                             | Value function being optimized (see [[1.0 - Value Function (M. Bellemare et al., 2017)]]).                                         |
| $\mathbb{E}\left[R(x, a)\right]$                      | Expected immediate reward from $(x, a)$.                                                                                           |
| $\gamma$                                               | Discount factor, $\gamma \in [0, 1]$.                                                                                              |
| $\mathbb{E}_P$                                         | Expectation over the transition kernel $P(\cdot \mid x, a)$ only (the policy is implicit through the $\max$).                      |
| $\max_{a' \in \mathcal{A}} Q(x', a')$                 | Greedy action selection at the next state $x'$: the best achievable value over all actions.                                         |
| $x'$                                                   | Next state, drawn from $P(\cdot \mid x, a)$.                                                                                       |
| $\mathcal{A}$                                          | Action space.                                                                                                                       |

The optimality operator $\mathcal{T}$ has a unique fixed point $Q^*$, corresponding to the set of optimal policies $\Pi^*$ (where $\pi^*$ is optimal if $\mathbb{E}_{a \sim \pi^*} Q^*(x, a) = \max_a Q^*(x, a)$). In the expected-value setting, $\mathcal{T}$ is a well-behaved $\gamma$-contraction.

The critical finding of this paper is that the distributional analogue — where the $\max$ selects a greedy policy and the full return distribution is propagated — is **not a contraction** in any distributional metric. While the expectations $\mathbb{E}[Z_k]$ still converge to $Q^*$ exponentially, the distributions themselves converge only weakly to the set of *nonstationary* optimal value distributions. This instability arises because a small change in the distribution can flip the greedy action, causing a large distributional shift.
